# 🌆 Urban Pulse

Urban Pulse is a modern web application built with React Router + Vite that provides a fast, responsive, and scalable user experience.

🌐 Live Demo: https://urban-pulse.vercel.app

---

## 🚀 Features

- ⚡ Fast Vite-powered frontend
- 📱 Fully responsive UI
- 🧭 React Router navigation
- ☁️ Deployed on Vercel with auto-deploy
- 🔗 GitHub integrated CI/CD

---

## 🛠 Tech Stack

- React / React Router
- Vite
- TypeScript / JavaScript
- Vercel (Hosting)
- GitHub (Version Control)

---
